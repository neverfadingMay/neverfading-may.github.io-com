adopt a new theme-tranquilpeak

发表于 2016-02-17   |     |   次阅读

本篇为我修改了主题为transquik

hexo 默认的主题是landscape,然后网上很多人推荐简洁的next主题，但是昨天闲的我又去翻了翻hexo的themes选择，发现一个挺中意的主题-tranquilpeak，接下来的都是蛮简单的，就是clone/或者就是下载zip，然后解压，就是按照tranquilpeak作者给写的quick start一步一步来，就可以了，当然花了很多时间在背景和头像上ORZ,其中有一些东西没有按照预期显示所以我又去看了看一个基于tranquilpeak的改编主题themia,themia做得也是很好看的，地址是themia的github地址,然后它的start quickly(?)也是写的很清楚，所以基本都是下载以后放在/hexo/themes里面，然后在/hexo/_config.yml中设置theme是xxxx(the name of the theme you adopt)。
嗯……花了一些时间在找cover和background images上，然后把头像换成了David.K.,这里不得不说我好像哪里出了问题所以我本地的照片显示不出来，所以我在网上找了图片，然后用external link顺利地显示出来了，头像的话有时候没有裁剪过显示出来的不是很好看，所以去大卫学习交流站找了一张接近于1*1的图片，而且像我现在在这里写markdown日记，它的本地上传还要钱，就是买会员一年99，可是宝宝穷（这并不是理由），所以我也是在网上找一张照片然后直接连过来，不知道会不会有额外损耗，然后我技术还没到能考虑这一步，有推荐说用七牛的图库，待会去试试。
全部改好以后就是

  • hexo g
  • hexo d
  • hexo s

就可以了。
我果然还是太花心思在装扮空间上了并且以为能够好好写日记，今天早上起来感觉不能再这样下去了，所以补上这篇，然后今天去走亲戚，回来看软件测试的pdf，有什么收获待会写上来
